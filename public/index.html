 Cara Deploy NFT Collection

        

          
1

          
Connect wallet EVM (MetaMask, Rabby, WalletConnect) lewat tombol di atas.

        

        

          
2

          
Pilih chain yang mau lo deploy. Pastikan wallet lo ada di network yang sama.

        

        

          
3

          
Isi data collection: nama, simbol, max supply, harga mint, dan batas mint per wallet.

        

        

          
4

          
Isi metadata: Hidden URI (placeholder sebelum reveal) dan Base URI (URL real metadata).

        

        

          
5

          
Klik Deploy Contract dan review transaksi di modal preview. Confirm di wallet lo.

        

        

          
6

          
Simpan alamat kontrak. Pakai tab Reveal untuk reveal collection di OpenSea.

 Flow Reveal Lengkap

        

          
1

          
Deploy kontrak ERC721 lo di tab Deploy. Set Hidden URI (placeholder metadata) dan Base URI (real metadata).

        

        

          
2

          
Mint — user mint NFT. Yang mereka lihat cuma placeholder art (hidden metadata).

        

        

          
3

          
Reveal — buka tab ini, call reveal() untuk switch dari hidden ke real metadata. Cuma deployer yang bisa.

        

        

          
4

          
Refresh — buka tab Refresh untuk trigger refresh metadata di OpenSea supaya collector lihat real artwork.

        

      


      

        
 Cara Reveal Collection

        

          
1

          
Connect wallet yang deploy kontrak. Cuma owner yang bisa reveal.

        

        

          
2

          
Pilih network yang sama tempat kontrak lo di-deploy.

        

        

          
3

          
Masukkan alamat kontrak (0x...) lalu klik Read Contract Status untuk cek status.

        

        

          
4

          
Masukkan Base URI baru — URL IPFS atau HTTPS yang menuju file metadata real (contoh: ipfs://QmXXX/).

        

        

          
5

          
Opsional: validasi URL metadata dengan klik Validate Metadata URLs untuk cek apakah file metadata bisa diakses.

        

        

          
6

          
Klik Confirm Reveal dan approve transaksi di wallet. Ini set baseURI real dan tandai collection sebagai revealed.

        

        

          
7

          
Setelah reveal, buka tab Refresh untuk refresh metadata di OpenSea supaya collector lihat real artwork.

 Cara Verifikasi Smart Contract

        

          
1

          
Masukkan alamat kontrak yang sudah lo deploy ke kolom di atas.

        

        

          
2

          
Pilih chain tempat kontrak lo di-deploy.

        

        

          
3

          
Klik Buka Etherscan / Blockscout. Lo akan dibawa ke halaman verifikasi di block explorer.

        

        

          
4

          
Isi form verifikasi:

            - Contract Name: NFTCollection

            - Compiler: 0.8.20

            - Optimization: Yes (200 runs)

            - License: MIT
          

        

        

          
5

          
Paste source code dari file contracts/NFTCollection.sol. Kalau pakai flatten, paste hasil flatten-nya.

        

        

          
6

          
Constructor arguments akan di-generate otomatis dari bytecode. Biasanya Etherscan bisa auto-detect.

        

        

          
7

          
Klik Verify & Publish. Proses butuh 1-2 menit. Setelah berhasil, kontrak lo akan punya badge verifikasi (centang biru) di Etherscan.

        

      


      

        
Kenapa Harus Verifikasi?

        

          
Trust & Transparansi — Kontrak yang terverifikasi bisa di-source code-nya oleh siapapun. User/collector bisa audit kode sebelum beli. Tanpa verifikasi, orang-orang ragu karena ga bisa cek apakah ada backdoor atau scam function.

        

        

          
Interaksi lebih mudah — Etherscan bisa auto-generate interface untuk call function (read/write). User bisa langsung mint, reveal, atau pause dari Etherscan tanpa perlu front-end.

        

        

          
OpenSea / Marketplace — Banyak marketplace verifikasi kontrak sebelum list. Kontrak yang ga verified seringkali ga muncul dengan benar.

 Cara Refresh Metadata di OpenSea

        

          
1

          
Paste URL OpenSea collection lo ke kolom di atas, lalu klik Parse Collection.

        

        

          
2

          
Klik link OpenSea untuk buka halaman collection. Kalau baru reveal, gambar mungkin masih placeholder.

        

        

          
3

          
Di halaman OpenSea, klik menu ••• (more options) di pojok kanan atas.

        

        

          
4

          
Pilih Refresh metadata dari dropdown. OpenSea akan queue refresh untuk collection lo.

        

        

          
5

          
Tunggu sekitar 5-10 menit untuk refresh selesai. Lo bisa cek dengan buka halaman token individual.

        

        

          
6

          
Refresh individual token: buka halaman token di OpenSea, klik menu •••, pilih Refresh metadata.
